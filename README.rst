Minter Name System
========

Minter Name System (MNS) is a Distributed Domain Name System by HashEx powered by Minter.

Features
--------

- Distribued and Secure Name System
- Interoperation with DNS
- Cryptoaddresses Resolver
- Decentralized Zones Support
